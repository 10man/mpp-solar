Python package with reference library of serial commands (and responses) 
for PIP-4048MS inverters - aka:
- PIP-4048MS 
- IPS-4000WM 
- Voltronic Axpert MKS 5KVA Off-grid Inverter-Chargers

Good overview and technical discussion here
